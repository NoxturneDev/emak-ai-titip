# Landing Page Specification

The landing page acts as the entry point for the **Emak AI Titip** platform, showcasing the value proposition of a traditional market jastip platform integrated with WhatsApp NLP parsing, SQLite transaction ledger, and secure Escrow management.

## 1. User Interface & Design System

The page is designed for high visual impact and maximum consistency with the pre-existing prototype control center:
* **Typography**: Outfit (headings) and Plus Jakarta Sans (body text).
* **Color Palette**: Dark background (`#0d0f14`), HSL-based Emerald Green primary theme (representing traditional fresh markets) and Sky Blue accents.
* **Layout**: Full responsiveness across mobile, tablet, and desktop screens with glassmorphic cards and dynamic glow elements.

## 2. Interactive Mockup Simulator

The hero section features a **Live AI Chatbot Parser Mockup** split-screen:
1. **WhatsApp Simulator (Left)**: Simulates typing of raw shopping lists (e.g. *"Beli tempe 1 papan, bayam 2 ikat, sama wortel 1/2 kg"*).
2. **JSON Parser Terminal (Right)**: Visualizes the structured JSON response generated by the AI parser, highlighting prices matched via fuzzy matching.

Pre-written templates are provided to let users witness real-time parsing capability instantly:
* *Sayur Lodeh*: "Beli tempe 1 papan, bayam 2 ikat, sama wortel 1/2 kg"
* *Sup Ayam*: "Tolong beliin ayam potong 1kg, bumbu racik sop 1, kentang 3 biji"
* *Santan Gula Merah*: "Minta tolong beliin kelapa parut 1 butir, gula merah 250gr"

## 3. Navigation & State Transitions

Transition between the marketing landing page and the interactive demo control center is managed through a state-driven router in `App.jsx`:
* **View State**: `view` state defaults to `'landing'`.
* **Launch Button**: Clicking "Mulai Demo Interaktif" transitions the view state to `'demo'`.
* **Return Link**: A "Kembali ke Beranda" button inside the control panel header transitions the state back to `'landing'`.
